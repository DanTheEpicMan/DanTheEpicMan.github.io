Day 3 - Loops and Strings

Loops are not that important for learing robotics, might be worth skipping

Strings store values:
std::string myString = "text";

Indexes

std::string myString = "text"; Analisis: t e x t 0 1 2 3 So doing: std::string charicter = myString[0]; means that charicter will be equal to "t" myString[1] is equal to e.

This string is, in essence, a list of charicters

Loops

Loops are used to repeat code

There are two main types of loops, the for loop and the while loop

While loops are used to loop until a condition is false

int i = 0; while (i < 10) { std::cout << i << " "; i = i + 1; }

This loop goes though and prints out the value of i followed by a space, the loop continues as long as i is under 10, i increases every loop by 1

The second type of loop, a for loop, can achive the same thing

for (int i = 0; i < 10; i++) { std::cout << i << " "; }

This loop does the same thing as the while loop above, but in a more compact way

In this loop, the first part (int i = 0) is the start of the loop, the second part (i < 10) is the condition that must be true for the loop to run, and the last part (i++) is what happens at the end of each loop
