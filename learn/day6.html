        Class Structure:

        class ClassName {
          private:
             //variables here. ie.
             int x;

          public:
            // constructor here. has the same name as the class, no return type

            ClassName(int var) {
              // set value of variable here, typucaly by using the passed in variable (peramiter)
            }

            ClassName() {
              //Shares the same name as the class but has no peramiter
              // this is a default constructor, in here your Instance variables (x), will be set to a default value
            }

            returnType functionName(peramiterType peramiterName) {
              // function code here
              return value; //must match the returnType
            }

            //Example
            int increaseAndReturn(int var) {
              x = x + var;
              return x;
            }
        }

        Vocab:
        Instance Variable: A variable that is declared (made) inside a class (not a function but the class itself)
        Declared: Made (example: int x = 0; is a declaration of a variable called x)
        Constructor: Called once when the class is creared
        Overload: A function or constructor that has the same name as another function or constructor but has different peramiters
        Peramiter: A variable that is passed into a function (example: int var in the function increaseAndReturn(int var))
        Return Type: The type of value that a function returns (example: int in the function increaseAndReturn(int var))
        Function: A block of code that can be called (example: int increaseAndReturn(int var) is a function)
        Function Call: When a function is called (example: increaseAndReturn(5) is a function call)
        Define: In the context of code, it means to write the code, opposite of Function Call
