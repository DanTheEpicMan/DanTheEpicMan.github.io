Day 1 - Variables

Variables are the most basic part of programming, they store values that can change at a later time

Variables in programming act similer to the way thet they do in math

The Box analogy

Variables act like a Box, each bag has a name, on the inside they can different values.
Depending on the Box type, only specific values can be stored.

In programming, we can create a box, give it a name, and put a value inside it.

Examples

There are many data types, here are two big ones.
int this type mean Integer and can only hold whole numbers
The second type if double, a double can hold decimal and hole numbers

You can change the value of code by using the =. Like this:

x = 4 (or, if x was a double, you could set it to a decimal value)

NOTE: In C++, the end of a statement (line of code) MUST have a semi-colin ;

What do you think is the result of setting the int variable to a decimal, try it out above.

Setting the value of variables to other variabels

An int and a double are equal to a number, so if you were to set a variable equal to another, or even itself, it would be set to that value

x = y;x is now equal to the value of y, so if y was 4, now x is 4

x = y + 5;in code, the side to the right of the equal sign is first evaluated (calculated to a number), then x is set to that value, in this case, if y was 4, the code would evaluate y + 5 to 9, and x would equal 9

Knowing that the right side is evaluated before that value is set to the variable, we can change the value of a variable relative to itself

The value of x equals 5, then this line is executed:
x = x + 3;
the value of x (5) is added to 3, and the new result is set as x

When y is 7, what do you think this line will equal?
y = y + 3;
y is now equal to 10, because the value of y (7) is added to 3, and the new result is set as y
