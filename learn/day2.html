Day 2 - True, False, and If statements

Another important type of variable is a boolean. A boolean is eather equal to true of false and nothing else. Example of definition

bool x = true;

Most of the time, boolean values arent hard coded (set in code to true or false), but rather functoins like the ones below evaluate (check) if a specific condition (expression/statement)

List of all of the oporators:

x == y if x is equal to y, returns true, else false
x != y if x is not equal to y, returns true, else false
x < y if x is less than y, returns true, else false
x > y if x is greater than y, returns true, else false
x <= y if x is less than or equal to y, returns true, else false
x >= y if x is greater than or equal to y, returns true, else false
Oporator Practice

Evaluate multiple conditions:

x && y if x and y are both true, returns true, else false, Example: x == 5 && y == 10
x || y if x or y are true, returns true, else false, Example: x == 5 || y == 10

If statements

An if statement is used to only execute a specific segment of code

if (boolean variable or statement) { //code to be executed }

Examples:

int x = 1; if (x == 1) { x = x + 3; //the value of x increases by 3 } x is now equal to 4

int x = 1; if (x == 0) { x = x + 3; //the value of x increases by 3 } x is value is unchanged and equals 1

because if statments only require a true of false, you can also pass in a boolean variable:

bool x = true; if (x) { x = x + 3; //the value of x increases by 3 } x is now equal to 4

The code in between the {} is a "layer" above the other code, if you were to declare a variable inside the {} and try to access it outside of the if statement, it would give you an error as that variable only exists inside the if statment

As you have already seen the inverse does not apply, variables outside can be access from the inside

Example that will throw an error:

int i = 1; if (i == 1) { int y = 2; }
This will throw an error

else statements

int x = 2; if (x == 1) { x = x + 3; //the value of x increases by 3 } else { x = x - 3; //the value of x decreases by 3 } x is now equal to -1

int x = 6; if (x == 1) { x = x + 3; //the value of x increases by 3 } else if (x > 3) { x = x - 3; //the value of x decreases by 3 } else { x = x * 3; //the value of x is multiplied by 3 } x is now equal to 3
