Day 4 - Classes

What if you could make your own variable type?

String Recap

std::string myName = "text";

What if we want to store info about a vehical, lets think about what attributes cars have: color, brand, distance traveled, number of wheels

Lets think about what data types these would use, color (std::string), brand (std::string), distance traveled (int), number of wheels (int)

We can use these basic variable types to construct a more complex one, here is an example:

class Car { std::string color; std::string brand; int distanceTraveled; int numberOfWheels; }

While all of the code we have written so far has taken place in the main() function, this will be outside of that

We now need to access these functions:

...Above class defined here... int main() { Car myCar; // Creates instance of the class with the name myCar myCar.color = "red"; myCar.brand = "Toyota"; myCar.distanceTraveled = 100; myCar.numberOfWheels = 4; // Access the attributes of the car std::cout << "Car color: " << myCar.color << ", Brand: " << myCar.brand << ", Distance traveled: " << myCar.distanceTraveled << ", Number of wheels: " << myCar.numberOfWheels << std::endl; }

As you can see, you access the attributes (variables), though the name of the class followed by a dot, followed by the name of the variable your trying to access
